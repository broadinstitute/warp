# What tags are included in an Optimus BAM file?

The Optimus Pipeline outputs a barcoded BAM file of aligned reads. There are multiple tags within the BAM file, including standard tags from 10X genomics (opens new window) and Sequence Alignment Map (SAM) files (opens new window). The table below details the tags used by the Optimus Pipeline and the relevant sources/tools from which the pipeline obtains the tags.

Optimus Pipeline BAM Tag Details Source
AS Alignment score generated by aligner SAM (opens new window)
CB Chromium cellular barcode sequence that is error-corrected and confirmed against a list of known-good barcode sequences 10X Genomics (opens new window)
CR Chromium cellular barcode sequence as reported by the sequencer 10X Genomics (opens new window)
CY Chromium cellular barcode read quality. Phred scores as reported by sequencer 10X Genomics (opens new window)
GE Gene exon Drop-seq Tools (opens new window)
GS Reserved for backwards compatibility reasons SAM (opens new window)
HI Query hit index SAM (opens new window)
MD String for mismatching indexes SAM (opens new window)
NH Number of reported alignments that contain the query in the current record SAM (opens new window)
NM Edit difference to the reference SAM (opens new window)
RG Read group SAM (opens new window)
UB Chromium molecular barcode sequence that is error-corrected among other molecular barcodes with the same cellular barcode and gene alignment 10X Genomics (opens new window)
UG Read group ID Umi-tools (opens new window)
UR Chromium molecular barcode sequence as reported by the sequencer 10X Genomics (opens new window)
UY Chromium molecular barcode read quality. Phred scores as reported by sequencer 10X Genomics (opens new window)
XF Extra alignment flags 10X Genomics (opens new window)
jI [jI:B:I,Start1,End1,Start2,End2,...] Start and End of introns for all junctions (1-based) STAR (opens new window)
jM [jM:B:c,M1,M2,...] Intron motifs for all junctions (i.e. N in CIGAR) STAR (opens new window)
nM The number of mismatches per (paired) alignment STAR (opens new window)
